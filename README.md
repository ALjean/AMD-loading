# POC Dynamic load script

## Getting Started

Every fragments app 
```
npm i
npm run build
```

Build shopping cart SDK

```
npm i && npm run build
```

Root folder

```
npm i
node index

http://localhost:8080/index
```

Use optimizer requirejs

```

npm install -g requirejs
r.js -o build.js optimize=none

```
